# Web demonstrator of the ODRL Validator

Web UI that demonstrates the [ODRL Validator](https://github.com/elenamolino/odrl-validator).

Features
- Text field to create ODRL policies
- Dropdown menu to select a set of pre-configured policies
- ODRL Validation: 
  - test for structural validation using [SHACL](https://www.w3.org/TR/shacl/) shapes
  - test for inconsistencies using [Notation3](https://w3c-cg.github.io/N3/spec/) Rules
- Human readable overview of the Violations, Warnings and conflicts
